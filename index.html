How to use it??

Download the source files, copy the rippllino.js file to your source folder. Now add a script tag in your html body. Then through in the value down below...
let rippllino = new Rippllino({ rplThemeColor: "#333333", rippllinoAnimateOn: "mousedown", animationDuration: 1 });

However, rplThemeColor, rippllinoAnimateOn, animationDuration can have custom user properties. 'rplThemeColor', can have hex color codes and rgb values. 'rippllinoAnimateOn' can have all mouseEvents in JavaScript


Link the file to your .html file. Now add a data-rippllino="true" on a button or a a tag. There it is, that should be working.

buttons / a tags sould have their own styles in order to make the rippll work!
